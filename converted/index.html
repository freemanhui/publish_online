We've added a feature that lets you publish your converted documents directly to this library:

  1. Convert your PDF using the PDF Converter
  2. Click the "Login with GitHub" button in the publishing section
  3. Use the simulation option to proceed with the demo
  4. Click "Publish to Repository" to prepare your document
  5. Download the HTML file and follow the instructions for uploading

In the future, this process will be fully automated with proper GitHub integration!

Manual Publishing Method

  1. Convert your PDF using the PDF Converter
  2. Download the HTML file when prompted
  3. Fork the repository
  4. Upload the HTML file to the converted/ folder in your fork
  5. Create a pull request, or if you're the site owner, commit directly to the repository
  6. Once merged, your document will be accessible at https://freemanhui.github.io/publish_online/converted/your-file-name.html

For site owners: Add new entries to this index.html file to list all available documents.
